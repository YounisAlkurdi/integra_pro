# 🛡️ Integra Smart Video Observer & Scrubbing Engine

توثيق تقني لآلية عمل "المراقب الذكي" ومحرك التحريك المعتمد على السكرول النسبي، لضمان أداء "Apple-Level" وحماية موارد الجهاز.

---

## 1. تقنية المراقب الذكي (Intersection Observer) 👁️
بدلاً من تشغيل محرك السكرول بشكل مستمر (والذي يستهلك طاقة المعالج حتى لو كان الفيديو غير ظاهر)، قمت بدمج **Intersection Observer**.

*   **آلية العمل:** يقوم النظام بمراقبة حاوية الفيديو (`.scroll-container`).
*   **التفعيل الذكي:** يتم تعيين الحالة `isActive: true` فقط عندما يدخل الفيديو في نطاق الرؤية (Viewport).
*   **الفائدة:** المحرك لا يحسب أي سكرول ولا يحرك أي فريم طالما أن المستخدم يقرأ النصوص العلوية أو السفلية.

```javascript
const observer = new IntersectionObserver((entries) => {
    this.state.isActive = entries[0].isIntersecting;
    // لا يبدأ التحريك إلا إذا كان الفيديو أمام المستخدم
    if (this.state.isActive) this.syncTarget();
}, { threshold: 0 });
```

---

## 2. عزل الأداء (Performance Isolation) ⚡
تضمن هذه التقنية عدم استهلاك موارد كرت الشاشة (GPU) أو المعالج (CPU) في العمليات الخلفية غير الضرورية.

*   **توقف Render Loop:** الحلقة البرمجية التي تقوم بعمل الـ Smoothing (LERP) تتوقف تماماً بمجرد خروج الفيديو من الشاشة.
*   **حماية الذاكرة:** يمنع النظام حدوث أي Memory Leaks أو عمليات إعادة رسم (Repaints) للفيديو وهو مخفي خلف النصوص.
*   **السلاسة:** عند الوصول للفيديو، تكون الموارد فارغة وجاهزة لتقديم 60 فريم في الثانية (60FPS) بدون أي تقطّع.

## 3. السكرول النسبي والدقيق (Relative Scrubbing) 📏
أكبر تحدي في سكرول الفيديو هو وجود محتوى طويل فوقه وتحته. التقليدي هو ربط الفيديو بكامل الصفحة، لكن في **Integra Engine**، السكرول "محلي" و "نسبي".

*   **نقطة البداية (Start Frame):** يبدأ الفريم الأول (00:00) تماماً عندما تلمس حاوية الفيديو أعلى الشاشة.
*   **نقطة النهاية (End Frame):** ينتهي الفيديو عند آخر فريم تماماً عند وصول المستخدم لنهاية مساحة السكرول المخصصة للفيديو.
*   **استقلالية المحتوى:** مهما زاد طول النصوص العلوية أو السفلية، يظل توقيت الفيديو متزامناً بدقة مع حركة يد المستخدم داخل قسم الفيديو فقط.

```javascript
// الحساب النسبي للموضع داخل الحاوية وليس الصفحة كاملة
const rect = this.container.getBoundingClientRect();
const scrollProgress = Math.abs(rect.top) / (containerHeight - viewHeight);
const clampedProgress = Math.max(0, Math.min(1, scrollProgress));
```

## 4. تجربة المستخدم (UX Strategy) 🚀
*   **بدون عشوائية:** لا يقفز الفيديو فجأة ولا يبدأ من المنتصف.
*   **تحكم كامل:** يشعر المستخدم أن إصبعة هو "عصا المايسترو" التي تحرك المشهد السينمائي.
*   **أمان الأداء:** ضمان عدم تعليق المتصفح (Browser Lag) حتى على الأجهزة المتوسطة.
